## 2. Mission & Vision Analysis

### 2.1 วิสัยทัศน์ BOI
> "Promotion of Investment to restructure Thailand's economy into new economy"
> (ส่งเสริมการลงทุนเพื่อปรับโครงสร้างเศรษฐกิจไทยสู่เศรษฐกิจใหม่)

### 2.2 ยุทธศาสตร์ 5 ปี (2023-2027)
3 เสาหลัก:
1. **Innovative** - เศรษฐกิจที่ขับเคลื่อนด้วยเทคโนโลยี นวัตกรรม และความคิดสร้างสรรค์
2. **Competitive** - เศรษฐกิจที่แข่งขันได้ ปรับตัวเร็ว สร้างการเติบโตสูง
3. **Inclusive** - เศรษฐกิจที่ยั่งยืน ลดความเหลื่อมล้ำ รักษาสิ่งแวดล้อม

### 2.3 ภารกิจหลัก
- ส่งเสริมให้เกิดการลงทุนในประเทศไทย
- ให้สิทธิประโยชน์และบริการอำนวยความสะดวกแก่นักลงทุนทั้งไทยและต่างประเทศ
- สนับสนุนนักลงทุนไทยที่ต้องการลงทุนในต่างประเทศ
- เป็นแหล่งข้อมูลสำหรับผู้บริหารและเจ้าหน้าที่ในการให้คำปรึกษาด้านการลงทุน

### 2.4 นัยยะต่อเว็บไซต์
เว็บไซต์ต้อง:
- สะท้อนภาพลักษณ์ **Innovative, Competitive, Inclusive**
- ต้องเป็น **"Digital Front Door"** ของ BOI สู่นักลงทุนทั่วโลก
- ต้องสื่อสาร **ความน่าเชื่อถือ ความเป็นมืออาชีพ** ขององค์กร
- ต้อง **ลดระยะห่าง** ระหว่างข้อมูลกับผู้ใช้ ให้เข้าถึงง่ายที่สุด

### 9.1 WCAG 2.1 Level AA Requirements (ตาม TOR §4.2.17)

#### Perceivable (รับรู้ได้)
- [ ] Text alternatives สำหรับ non-text content (alt text ทุกรูป)
- [ ] Captions สำหรับ multimedia (video ทุกตัว)
- [ ] Content สามารถนำเสนอได้หลายรูปแบบ
- [ ] Color contrast ratio >= 4.5:1 (normal text), >= 3:1 (large text)
- [ ] Text resize ได้ถึง 200% โดยไม่สูญเสียข้อมูล
- [ ] ไม่ใช้สีเพียงอย่างเดียวในการสื่อความหมาย

- [ ] ไม่มี content ที่ทำให้ชัก (flash < 3 times/sec)
- [ ] Skip navigation links
- [ ] Focus visible ทุก interactive element

#### Understandable (เข้าใจได้)
- [ ] ภาษาของหน้าระบุชัดเจน (lang attribute ตาม locale)
- [ ] Navigation consistent ทุกหน้า
- [ ] Error identification ชัดเจน — ระบุจุดที่ผิดในฟอร์ม
- [ ] Labels/Instructions สำหรับ input (htmlFor/id เชื่อมถูกต้อง)

#### Robust (แข็งแกร่ง)
- [ ] Valid HTML markup (ผ่าน validator.w3.org — 0 errors)
- [ ] Valid CSS (ผ่าน jigsaw.w3.org/css-validator — 0 errors)
- [ ] ARIA labels ที่เหมาะสม
- [ ] Compatible กับ assistive technologies (screen readers)

### 13.3 Anti-Killer Page Strategy

3. **Next/Previous** - สำหรับเนื้อหาที่เป็น series
4. **CTA (Call-to-Action)** - ทุกหน้าต้องมี CTA ชัดเจน
5. **Tag System** - ระบบ tag เชื่อมโยงเนื้อหาข้ามหมวด
6. **Cross-linking** - ลิงก์ภายในเนื้อหาไปยังหน้าที่เกี่ยวข้อง
7. **Sidebar** - Quick links, Popular content, Recent updates

## 15. SWOT Analysis

### 15.1 Strengths (จุดแข็ง)
- BOI เป็น single authority ด้านส่งเสริมการลงทุน - ข้อมูลเป็น "Source of Truth"
- มีระบบ e-service หลายตัว (e-Investment, VISA, Smart Visa, LTR, OSOS)
- มีข้อมูลสถิติการลงทุนครบถ้วน
- รองรับ 7 ภาษา
- มีสำนักงานทั้งในไทยและต่างประเทศ
- มี Success Stories จริง

### 15.2 Weaknesses (จุดอ่อน)
- เทคโนโลยีเก่า (PHP legacy) ถูก attack บ่อย
- URL structure ไม่ friendly ต่อ SEO
- Mobile experience ไม่ดี
- Navigation ซับซ้อน (4+ levels)
- Sub-sites กระจัดกระจาย (8+ portals)
- CMS ใช้ยาก เจ้าหน้าที่อัปเดตเนื้อหาช้า
- ไม่มี interactive tools (eligibility checker, calculator)
- PDF-heavy ไม่มี web version
- Performance ช้า ไม่ผ่าน Core Web Vitals
- ไม่มี Related Content / Cross-linking

### 15.3 Opportunities (โอกาส)
- Next.js จะทำให้เว็บเร็วขึ้นมาก (SSR/SSG)
- AI Search optimization เป็นโอกาสใหม่ที่คู่แข่งยังไม่ทำ
- Headless CMS จะทำให้เจ้าหน้าที่อัปเดตเนื้อหาง่าย
- รวม Sub-sites ภายใต้ UX เดียวกัน ลดความสับสน
- Interactive tools จะ differentiate จากเว็บ IPA อื่นๆ ในภูมิภาค
- Progressive Web App (PWA) สำหรับ offline access
- Personalization ตามกลุ่มผู้ใช้
- Data Dashboard สำหรับสถิติการลงทุน

### 15.4 Threats (ภัยคุกคาม)
- Cyber attacks - ต้อง robust ทั้ง OWASP และ Penetration Testing
- เทคโนโลยีเปลี่ยนเร็ว - ต้อง scalable architecture
- คู่แข่ง (IPA ประเทศอื่น) ปรับปรุงเว็บเร็ว
- WCAG/W3C standards เปลี่ยนแปลง
- งบประมาณจำกัด เทียบกับขอบเขตงานที่กว้าง
